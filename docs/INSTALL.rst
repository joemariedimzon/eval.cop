eval.cop Installation
---------------------

To install eval.cop using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``eval.cop`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        eval.cop

* Re-run buildout, e.g. with:

    $ ./bin/buildout
